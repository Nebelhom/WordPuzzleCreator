About this site

Using this site, you will be able to create various wordpuzzles and save them as XLS files. Currently, only the popular word search is available, but there is a crossword puzzle creator in planning.

This site is written using the popular flask microframework together with the python programming language. Additionally, a bit of jquery made the site more accessible.
